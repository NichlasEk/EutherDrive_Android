EutherDrive environment variables (EUTHERDRIVE_*).

General
- EUTHERDRIVE_DUMP_VECTORS=1: dump initial vector bytes on load.
- EUTHERDRIVE_FB_TRACE=1: framebuffer identity/sample tracing.
- EUTHERDRIVE_TRACE_PERF=1: perf timers/hotspots logging (CPU/VDP/blit).
- EUTHERDRIVE_PERF_STATS=1: bus perf stats.
- EUTHERDRIVE_REGION=auto|jp|us|eu: force console region (defaults to auto).
- EUTHERDRIVE_TRACE_IO=1: log IO reads (0xA10001, 0xA10003..0xA1001F).
- EUTHERDRIVE_HEADLESS_DUMP_DIR=<path>: override headless frame dump directory (default: ROM directory).

UI
- EUTHERDRIVE_UI_HEARTBEAT=1: UI heartbeat log.
- EUTHERDRIVE_TRACE_TICK=1: per-tick log.
- EUTHERDRIVE_TRACE_UI_FRAME=1: frame-level UI logs (heartbeat ticks, basic frame flow).
- EUTHERDRIVE_TRACE_UI_RENDER=1: UI render/emu-loop debug logs (bitmap recreate, emu-loop exceptions).
- EUTHERDRIVE_TRACE_UI_PRESENT=1: per-present logs (frame id, empty frame, present metrics).
- EUTHERDRIVE_TRACE_UI_AUDIO=1: show audio debug line in UI (buffer, accumulator, timing ratio).
- EUTHERDRIVE_TRACE_Z80_YM_ALL=1: log all Z80 YM2612 writes (very noisy; gated).
- EUTHERDRIVE_TRACE_Z80_YM_ALL_LIMIT=<int>: max Z80 YM write logs (default 256; 0=unlimited).
- EUTHERDRIVE_TRACE_VDP_CTRL=1: log VDP control-port writes/decodes (very noisy; gated).
- EUTHERDRIVE_TRACE_VDP_CTRL_LIMIT=<int>: max VDP ctrl logs (default 200; 0=unlimited).
- EUTHERDRIVE_TRACE_VDP_RENDER=1: log VDP render debug markers (very noisy; gated).
- EUTHERDRIVE_TRACE_VRAM_WRITE_DETAIL=1: log detailed VRAM writes (very noisy; gated).
- EUTHERDRIVE_TRACE_VDP_IO=1: log raw VDP IO writes (very noisy; gated).
- EUTHERDRIVE_TRACE_VDP_IO_LIMIT=<int>: max VDP IO logs (default 200; 0=unlimited).
- EUTHERDRIVE_TRACE_VRAM_WRITE_CPU=1: log CPU VRAM writes (very noisy; gated).
- EUTHERDRIVE_TRACE_VRAM_WRITE_CPU_LIMIT=<int>: max CPU VRAM write logs (default 256; 0=unlimited).
- EUTHERDRIVE_TRACE_VDP_CTRL_WRITE=1: log key VDP control register writes (gated).
- EUTHERDRIVE_TRACE_VDP_CTRL_WRITE_LIMIT=<int>: max VDP ctrl write logs (default 128; 0=unlimited).
- EUTHERDRIVE_TRACE_VDP_ADDR_SET=1: log VDP address set commands (gated).
- EUTHERDRIVE_TRACE_VDP_ADDR_SET_LIMIT=<int>: max VDP addr logs (default 128; 0=unlimited).
- EUTHERDRIVE_TRACE_BUS_VDP=1: log VDP writes at bus layer (gated).
- EUTHERDRIVE_TRACE_BUS_VDP_LIMIT=<int>: max VDP bus logs (default 200; 0=unlimited).
- EUTHERDRIVE_TRACE_Z80SAFE=1: log Z80 safe-boot tick/debug (gated).
- EUTHERDRIVE_TRACE_MADOU=1: log Madou trace points (gated).
- EUTHERDRIVE_TRACE_MADOU_FULL=1: log extended Madou trace points (very noisy; gated).
- EUTHERDRIVE_TRACE_ALADDIN_DEBUG=1: log per-frame Z80/audio debug counters (gated).
- EUTHERDRIVE_TRACE_DMA_REGS=1: log VDP DMA register updates and DMA start/copy details (very noisy; gated).
- EUTHERDRIVE_TRACE_VDP_FRAME=1: log VDP per-frame summary (gated).
- EUTHERDRIVE_TRACE_VINT=1: log VBlank interrupts (gated).
- EUTHERDRIVE_TRACE_Z80_FORCE_ACTIVE=1: log when Z80 is forced active (gated).
- EUTHERDRIVE_TRACE_Z80_INT_DEBUG=1: log Z80 interrupt debug lines (gated).
- EUTHERDRIVE_TRACE_Z80_BOOT=1: log Z80 boot/driver entry dumps (very noisy; gated).
- EUTHERDRIVE_TRACE_INTERLACE_DEBUG=1: log interlace debug stats (very noisy; gated).
- EUTHERDRIVE_TRACE_NAME_TABLE=1: log name table summary at VBlank (gated).
- EUTHERDRIVE_TRACE_VRAM_PAGE_STATS=1: log VRAM page stats at VBlank (gated).
- EUTHERDRIVE_TRACE_VRAM_STATS=1: log per-frame VRAM stats (gated).
- EUTHERDRIVE_AUDIO=1: enable audio output (pw-cat).
- EUTHERDRIVE_AUDIO_TIMED=1: time-based audio fill (wall-clock) to keep audio steady when FPS is low (default on; set 0 to disable).
- EUTHERDRIVE_AUDIO_TIMED_MAX_FRAMES=<int>: cap frames generated per tick in timed mode (default 1024).
- EUTHERDRIVE_AUDIO_TIMED_DRAIN=1: pace audio drain on wall-clock (default off).
- EUTHERDRIVE_AUDIO_OUT_PLL=1: enable output-side PLL resampling to keep buffer near target (default off).
- EUTHERDRIVE_AUDIO_OUT_PLL_MAX=<float>: max output PLL ratio adjustment (default 0.005).

VDP
- EUTHERDRIVE_TRACE_VDP=1: VDP log (general).
- EUTHERDRIVE_TRACE_MD_VDP=1: MD VDP log (core).
- EUTHERDRIVE_TRACE_VDP_TIMING=1: VDP timing log.
- EUTHERDRIVE_TRACE_VDP_FB=1: VDP framebuffer sampling.
- EUTHERDRIVE_TRACE_VDP_STATUS_READ=1: log VDP status reads (C00004) with vblank/overflow/collision bits.
- EUTHERDRIVE_TRACE_SPRITE_OVERFLOW_FRAME=1: log per-frame max sprites/cells per line + overflow flag.
- EUTHERDRIVE_TRACE_VRAM=1: VRAM vs renderer VRAM sampling.
- EUTHERDRIVE_TRACE_VRAM_RANGE=<start-end>: log VRAM writes in range (hex; e.g. 0xC000-0xC7FF).
- EUTHERDRIVE_TRACE_VRAM_RANGE_LIMIT=<int>: limit [VRAM-RANGE] logs (0 = unlimited; default 200).
- EUTHERDRIVE_TRACE_VRAM_RANGE_SKIP_FILL=1: skip [VRAM-RANGE] logs for DMA-FILL writes.
- EUTHERDRIVE_TRACE_DMA_SRC=1: log DMA source reads (address/value) for VDP DMA.
- EUTHERDRIVE_TRACE_DMA_SRC_LIMIT=<int>: limit [DMA-SRC-READ] logs (0 = unlimited; default 128).
- EUTHERDRIVE_TRACE_TILE_FETCH=1: log tile fetches in renderer (per scanline).
- EUTHERDRIVE_TRACE_TILE_FETCH_SCANLINE=<int>: scanline to log (default 112).
- EUTHERDRIVE_TRACE_TILE_FETCH_LIMIT=<int>: max [TILEFETCH] logs per frame (0 = unlimited; default 128).
- EUTHERDRIVE_TRACE_SCROLL_LINE=1: log hscroll/vscroll sampling for a scanline.
- EUTHERDRIVE_TRACE_SCROLL_LINE_SCANLINE=<int>: scanline to log (default 112).
- EUTHERDRIVE_TRACE_SCROLL_LINE_LIMIT=<int>: max [SCROLLLINE] logs per frame (0 = unlimited; default 32).
- EUTHERDRIVE_FORCE_SCROLL_ZERO=1: force H/V scroll to 0 during rendering (debug).

RAM
- EUTHERDRIVE_TRACE_RAM_RANGE=<start-end>: log RAM reads/writes in range (hex; e.g. 0xFF0000-0xFF0FFF).
- EUTHERDRIVE_TRACE_RAM_RANGE_LIMIT=<int>: limit [RAM-RANGE] logs (0 = unlimited; default 200).
- EUTHERDRIVE_TRACE_RAM_RANGE_NONZERO=1: only log non-zero RAM range values.
- EUTHERDRIVE_TRACE_RAM_RANGE_FIRST_PER_FRAME=1: only log first matching RAM range event per frame.
- EUTHERDRIVE_TRACE_RAM_RANGE_WRITE_COUNTER=1: count RAM writes in range per frame and log a summary.
- EUTHERDRIVE_TRACE_RAM_RANGE_FIRST_WRITE=1: log first write per frame in range (even if value is 0).
- EUTHERDRIVE_TRACE_CRAM=1: CRAM write log.
- EUTHERDRIVE_TRACE_NAMETABLE_ROW_DUMP=1: dump Plane A/B nametable rows + key VDP regs (scanline 0 and 112).
- EUTHERDRIVE_TRACE_PATTERN_TILE_DUMP=1: dump 8x8 tile patterns for first few nametable entries.
- EUTHERDRIVE_FORCE_MD_VBLANK=1: force MD vblank.
- EUTHERDRIVE_FORCE_SMS_VBLANK=1: force SMS vblank.

CPU / 68k
- EUTHERDRIVE_TRACE_MD_STALL=1: stall watchdog log.
- EUTHERDRIVE_TRACE_PC=1: PC watch logs.
- EUTHERDRIVE_TRACE_PC_TAP_LIST=<addr,addr,...>: log regs + disasm when PC hits any listed address.
- EUTHERDRIVE_TRACE_PC_TAP_PEEK_LIST=<start-end,...>: dump RAM bytes for listed ranges on PC tap.
- EUTHERDRIVE_TRACE_PC_TAP_ONCE_PER_FRAME=1: log at most once per frame per tapped PC.
- EUTHERDRIVE_TRACE_DBRA=1: DBRA trace.
- EUTHERDRIVE_TRACE_TRAP=1: TRAP trace.
- EUTHERDRIVE_TRACE_CALLPARAM=1: log stack argument near 0x02EB60..0x02EB80.
- EUTHERDRIVE_TRACE_MOVEA_PARAM=1: log MOVEA.L (A7+8) param near 0x02EB80..0x02EBC0.
- EUTHERDRIVE_TRACE_ROMCOPY=1: ROM copy trace.
- EUTHERDRIVE_TRACE_RESETVECT=1: reset vector trace.
- EUTHERDRIVE_TRACE_MEM_WATCH_LIST=<addr,addr,...>: log reads/writes to any listed address (hex or dec).
- EUTHERDRIVE_FORCE_B154_READ=<byte>: force 68k reads at 0xFFB154 to this value (hex or dec).
- EUTHERDRIVE_FORCE_B154_READ_LIMIT=<int>: limit [B154-OVERRIDE] logs (0 = unlimited).

WRAM
- EUTHERDRIVE_TRACE_WRAM=1: WRAM writes (sampled).
- EUTHERDRIVE_TRACE_WRAMTABLE=1: WRAM table range logs (0xFF0540..0xFF0580).
- EUTHERDRIVE_TRACE_WRAMSLOTS=1: WRAM slot writes (0xFF0556..0xFF055D).

Z80
- EUTHERDRIVE_TRACE_Z80=1: Z80 trace.
- EUTHERDRIVE_TRACE_Z80SIG=1: Z80 busreq/busack signal logs.
- EUTHERDRIVE_TRACE_Z80WIN=1: Z80 window bus logs.
- EUTHERDRIVE_TRACE_Z80WIN_LIMIT=<int>: limit Z80 window bus logs (0 = unlimited).
- EUTHERDRIVE_Z80_BANK_DEFAULT=<int|hex>: default Z80 bank register (9-bit).
- EUTHERDRIVE_FORCE_SMS_STATUS7=1: force SMS status bit 7.
- EUTHERDRIVE_TRACE_BOOT=1: Z80 boot trace.
- EUTHERDRIVE_TRACE_SMS_DELAY=1: SMS delay trace.
- EUTHERDRIVE_SKIP_SMS_DELAYS=1: skip SMS delays.
- EUTHERDRIVE_Z80_RESET_HOLD_Z80_CYCLES=<int>: hold Z80 execution for N cycles after reset deassert.
- EUTHERDRIVE_Z80_RESET_HOLD_LIMIT=<int>: limit [Z80RESET-HOLD] logs (0 = unlimited).
- EUTHERDRIVE_Z80_SAFE_BOOT=1|0: enable/disable Z80 safe boot (default: 0/off).
- EUTHERDRIVE_Z80FLAG65_READ_FROM_MBX=1: override Z80 reads at 0x0065 from mailbox RAM (read-only).
- EUTHERDRIVE_Z80FLAG65_READ_ADDR=<hex|dec>: Z80 flag addr to override (default 0x0065).
- EUTHERDRIVE_Z80FLAG65_READ_SOURCE=<hex|dec>: source mailbox addr in Z80 RAM (default 0x1B8F).
- EUTHERDRIVE_Z80FLAG65_READ_AND=<hex|dec>: optional AND mask applied to the source value.
- EUTHERDRIVE_Z80FLAG65_READ_OR=<hex|dec>: optional OR mask applied to the source value.
- EUTHERDRIVE_Z80FLAG65_READ_LIMIT=<int>: limit [Z80FLAG65-READ] logs (0 = unlimited).
- EUTHERDRIVE_Z80_BOOT_JP=1: patch Z80 RAM at 0x0000 with `JP <target>` on reset.
- EUTHERDRIVE_Z80_BOOT_JP_TARGET=<hex|dec>: boot jump target (default 0x0D00).
- EUTHERDRIVE_Z80_BOOT_JP_LIMIT=<int>: limit [Z80BOOT-JP] logs (0 = unlimited).
- EUTHERDRIVE_Z80FLAG_FORCE_JR=1: force JR path after DDCB BIT at 0x0DD8 when bit2 is set.
- EUTHERDRIVE_Z80FLAG_FORCE_JR_TARGET=<hex|dec>: override forced JR target (default uses the JR+disp at 0x0DDC).
- EUTHERDRIVE_Z80FLAG_FORCE_JR_LIMIT=<int>: limit [Z80FLAG-JR] logs (0 = unlimited).
- EUTHERDRIVE_Z80_RESET_ON_BUSREQ_RELEASE=1: reset Z80 once when BUSREQ deasserts (debug).
- EUTHERDRIVE_Z80_RESET_ON_BUSREQ_RELEASE_LIMIT=<int>: limit [Z80BUSREQ-RESET] logs/resets (0 = unlimited).
- EUTHERDRIVE_Z80_FORCE_PC_ON_UPLOAD=1: force Z80 PC to target after a 68k upload write hits the Z80 window range.
- EUTHERDRIVE_Z80_FORCE_PC_ON_UPLOAD_START=<hex|dec>: range start (default 0x0D00).
- EUTHERDRIVE_Z80_FORCE_PC_ON_UPLOAD_END=<hex|dec>: range end (default 0x0E50).
- EUTHERDRIVE_Z80_FORCE_PC_ON_UPLOAD_TARGET=<hex|dec>: forced PC target (default 0x0D00).
- EUTHERDRIVE_Z80_FORCE_PC_ON_UPLOAD_LIMIT=<int>: limit [Z80PC-ARM] logs (0 = unlimited).
- EUTHERDRIVE_Z80_VBLANK_IRQ_PULSE=<int>: auto-clear Z80 VBlank IRQ after N Z80 cycles (default 171).
- EUTHERDRIVE_Z80_YM_WAIT=<int>: extra Z80 wait cycles for YM2612 port access (default 4).
- EUTHERDRIVE_Z80_PSG_WAIT=<int>: extra Z80 wait cycles for PSG writes (default 4).
- EUTHERDRIVE_Z80_BANK_WAIT=<int>: extra Z80 wait cycles for 68k banked reads/writes (default 16).
- EUTHERDRIVE_TRACE_Z80BANK_READ_LIMIT=<int>: limit [Z80BANKRD] logs (0 = unlimited).
- EUTHERDRIVE_TRACE_Z80_FRAME_CYCLES=1: log Z80 actual/budget cycle stats per frame.
- EUTHERDRIVE_TRACE_Z80_FRAME_CYCLES_EVERY=<int>: frame interval for Z80 cycle logs (default 60).
- EUTHERDRIVE_TRACE_Z80_AUDIO_RATE=1: log YM status read/PSG write rates from Z80 cycles.
- EUTHERDRIVE_TRACE_Z80_AUDIO_RATE_EVERY=<int>: frame interval for audio rate logs (default 60).
- EUTHERDRIVE_TRACE_Z80_AUDIO_RATE_START_FRAME=<int>: delay audio rate logs until this frame (default 0).
- EUTHERDRIVE_TRACE_YM_TIMER=1: log YM2612 timer register writes and per-frame timer event counts.
- EUTHERDRIVE_TRACE_YM_TIMER_LIMIT=<int>: limit [YMTIMER-REG] logs (default 256).
- EUTHERDRIVE_TRACE_YM_KEY=1: log YM2612 key on/off writes and per-frame key stats.
- EUTHERDRIVE_TRACE_YM_KEY_LIMIT=<int>: limit [YMKEY] logs (default 256).
- EUTHERDRIVE_TRACE_YM_WRITE_STATS=1: log YM2612 per-frame write counts and last register write.
- EUTHERDRIVE_TRACE_YM_WRITE_STATS_LIMIT=<int>: limit [YMWRITE] logs (default 256).
- EUTHERDRIVE_TRACE_Z80_PC_RANGE_START=<hex|dec>: log Z80 opcodes in this PC range (start).
- EUTHERDRIVE_TRACE_Z80_PC_RANGE_END=<hex|dec>: log Z80 opcodes in this PC range (end).
- EUTHERDRIVE_TRACE_Z80_PC_RANGE_LIMIT=<int>: limit [Z80PC] logs (0 = unlimited).
- EUTHERDRIVE_FORCE_Z80_B154_MAP=1: map Z80 reads/writes at 0xB154 to a fixed 68k address.
- EUTHERDRIVE_FORCE_Z80_B154_ADDR=<hex|dec>: target 68k addr (default 0xFFB154).
- EUTHERDRIVE_FORCE_Z80_B154_Z80ADDR=<hex|dec>: Z80 addr to intercept (default 0xB154).
- EUTHERDRIVE_FORCE_Z80_B154_Z80ADDR2=<hex|dec>: optional second Z80 addr to intercept.
- EUTHERDRIVE_FORCE_Z80_B154_LIMIT=<int>: limit [Z80B154-OVERRIDE] logs (0 = unlimited).
- EUTHERDRIVE_MIRROR_MBX_TO_Z80FLAG_RANGE=1: mirror 0x1B80..0x1B8F into Z80 RAM (default 0x0060..0x006F).
- EUTHERDRIVE_MIRROR_MBX_TO_Z80FLAG_RANGE_ADDR=<hex|dec>: base address for the mirror range (default 0x0060).
- EUTHERDRIVE_MIRROR_MBX_TO_Z80FLAG_RANGE_LIMIT=<int>: limit [Z80FLAG-RANGE] logs (0 = unlimited).
- EUTHERDRIVE_MIRROR_MBX_WIDE_TO_Z80FLAG_RANGE=1: mirror a sub-range of 0x1B00..0x1B7F into Z80 RAM.
- EUTHERDRIVE_MIRROR_MBX_WIDE_TO_Z80FLAG_RANGE_START=<hex|dec>: source start (default 0x1B20).
- EUTHERDRIVE_MIRROR_MBX_WIDE_TO_Z80FLAG_RANGE_END=<hex|dec>: source end (default 0x1B2F).
- EUTHERDRIVE_MIRROR_MBX_WIDE_TO_Z80FLAG_RANGE_ADDR=<hex|dec>: destination base (default 0x0060).
- EUTHERDRIVE_MIRROR_MBX_WIDE_TO_Z80FLAG_RANGE_LIMIT=<int>: limit [Z80FLAG-WIDE] logs (0 = unlimited).

Pads
- EUTHERDRIVE_TRACE_PAD=1: pad read/write trace.
- EUTHERDRIVE_PAD_SELFTEST=1: pad self-test logs.
- EUTHERDRIVE_PAD_TYPE=ThreeButton|SixButton: pad type override.

Audio / PSG / YM
- EUTHERDRIVE_TRACE_AUDIO=1: audio min/max log (per second).
- EUTHERDRIVE_TRACE_AUDLVL=1: audio level/peak logs.
- EUTHERDRIVE_TRACE_PSG=1: PSG write log.
- EUTHERDRIVE_DISABLE_PSG=1: disable PSG mix.
- EUTHERDRIVE_GEMS_TIMING=1: enable GEMS timing mode (negative audio accumulator for GEMS games).
- EUTHERDRIVE_AUDIO_ACCUM_FRAMES=<float>: override initial audio accumulator in frames (universal; overrides GEMS timing).
- EUTHERDRIVE_YM_RESAMPLE=linear|cubic: select YM resampler (default cubic).
- EUTHERDRIVE_YM_RESAMPLE_LINEAR=1: alias for linear resampler.
- EUTHERDRIVE_YM=1: enable YM2612 access/mix.
- EUTHERDRIVE_TRACE_YM=1: YM write log (first N writes).

SMS timing
- EUTHERDRIVE_SMS_CYCLES_MULT=<int>: SMS cycles multiplier.

MD timing
- EUTHERDRIVE_Z80_CYCLES_MULT=<float>: Z80 cycles multiplier.

Bus
- EUTHERDRIVE_TRACE_BUS=1: bus address routing trace.

Stubs enabled by default
- TRAP stub at 0xFF0500 (RTS) is installed and pointers 0xFF0556/0xFF055A/0xFF0562/0xFF0566/0xFF056A are set to it.
  Disable with EUTHERDRIVE_TRAP_STUB=0.
